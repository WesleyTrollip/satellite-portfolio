# Satellite Portfolio Business Requirements

## Purpose

The satellite portfolio system supports:

-   Experimental investing strategies
-   AI assisted analysis
-   Portfolio monitoring
-   Risk management enforcement

## Portfolio Size

Initial capital: €30,000

## Allowed Strategy Types

-   Individual stocks
-   Sector ETFs
-   Macro‑driven trades
-   AI‑assisted strategies
-   Thematic investments

Examples: - SGHC - Semiconductor ETFs - AI sector ETFs

## Risk Limits

  Rule                 Limit
  -------------------- -------------
  Max position size    15%
  Max portfolio loss   50%
  Leverage             Not allowed
  Options trading      Not allowed

## Portfolio Segmentation

  Category            Description
  ------------------- --------------------------
  Core experimental   medium conviction trades
  Speculative         high risk experiments
  AI strategy         algorithmic signals

## Required System Features

The system must support:

-   Portfolio tracking
-   Trade logging
-   Position analysis
-   Risk monitoring
-   AI research support

## Alerting Requirements

Alerts must trigger when:

-   Position exceeds allowed size
-   Portfolio drawdown \>20%
-   Portfolio drawdown \>30%
-   Asset concentration becomes excessive
